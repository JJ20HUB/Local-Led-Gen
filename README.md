## Fastest Way to Get an Active URL + Domain

### Option A (Recommended): Cloudflare Pages + Custom Domain

1. Push this repo to GitHub.
2. In Cloudflare Pages, create a new project from this repo.
3. Build command: leave empty (static site).
4. Output directory: `/` (root).
5. Deploy.
6. Add your custom domain in Cloudflare Pages (for example `treeprosconnect.com`).
7. Set DNS records as prompted.
8. Enable automatic HTTPS.

You will get an active URL immediately (Pages URL), then your branded domain once DNS propagates.

### Option B: WordPress Hosting (if you want WP)

If you specifically want WordPress:

1. Buy hosting that includes WordPress (SiteGround, WP Engine, etc.).
2. Point your domain DNS to that host.
3. Install WordPress.
4. Recreate these 3 pages in WordPress (Home, Services, Contact).
5. Add click-to-call links using `tel:+1XXXXXXXXXX` in buttons.
6. Install a form plugin (WPForms or Fluent Forms) for lead capture.
7. Submit your final live URL to your Google Business Profile website field.
